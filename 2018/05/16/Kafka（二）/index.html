概述

kafka是一种分布式、高吞吐量的消息中间件，通过磁盘数据结构来提供消息的持久化。
本文主要记录使用java操作kafka生产者与消费者。

kafka

此版本的spring boot支持自动配置kafka，application.properties中使用前缀为spring.kafka.* 的属性，不过本文中没有使用。
kafka安装的版本是kafka_2.11-0.11.0.2.tgz

源码分析

ConcurrentMessageListenerContainer

ConcurrentMessageListenerContainer有三个属性
ConsumerFactory<K, V> consumerFactory; List<KafkaMessageListenerContainer<K, V>> containers = new ArrayList(); int concurrency = 1;
ConcurrentMessageListenerContainer通过doStart()方法并使用consumerFactory创建concurrency个KafkaMessageListenerContainer，并调用其start()方法，实际上执行了KafkaMessageListenerContainer的doStart()方法。

KafkaMessageListenerContainer
